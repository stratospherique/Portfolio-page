## 🚀 Getting Started

To get a local copy up and running follow these simple example steps.

### Clone

```sh
git clone git@github.com:stratospherique/Portfolio-page.git
cd Portfolio-page
```

### Install

```sh
npm install
```

### Usage

```sh
npm start
```
